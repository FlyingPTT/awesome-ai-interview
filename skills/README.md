# AI Skills Collection

This directory collects reusable AI skills and workflows. A skill is a compact instruction package that helps an AI agent perform a repeatable task reliably.

## Skill Categories

- [Research Skills](research-skills.md)
- [Coding Agent Skills](coding-agent-skills.md)
- [Agent Skills Catalog](agent-skills-catalog.md)
- [Interview Skills](interview-skills.md)
- [RAG Diagnosis Skills](rag-diagnosis-skills.md)

## What Makes A Good Skill

A good skill should define:

- Purpose.
- Inputs.
- Tools.
- Workflow.
- Quality bar.
- Failure handling.
- Output format.

## Skill Template

```md
# Skill Name

## Purpose

Describe the repeatable task.

## Inputs

- Input 1.
- Input 2.

## Workflow

1. Inspect context.
2. Identify constraints.
3. Execute the task.
4. Verify the output.
5. Report the result.

## Quality Bar

- Requirement 1.
- Requirement 2.

## Output Format

Define the final deliverable.
```
