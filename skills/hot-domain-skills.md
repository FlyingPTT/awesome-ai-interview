# Hot Domain Skills

These skills are useful for following fast-moving AI topics and turning them into interview-ready notes.

## Weekly AI Trend Digest Skill

Purpose: summarize a week's AI developments into actionable interview and project notes.

Workflow:

1. Collect model releases, papers, frameworks, and product changes.
2. Group them by theme.
3. Identify why each item matters.
4. Extract interview questions.
5. Extract project ideas.

Output:

- Top trends.
- Key papers.
- Tools to try.
- Interview talking points.
- Project opportunities.

## Hot Topic To Project Skill

Purpose: convert a trending AI topic into a portfolio project.

Workflow:

1. Identify the core technical idea.
2. Pick a small but realistic use case.
3. Design a minimal architecture.
4. Define metrics.
5. Write resume bullets.
6. List interview questions.

Example topics:

- Agentic RAG.
- AI coding agents.
- Multimodal document agents.
- LLM evaluation platforms.
- Synthetic data pipelines.
