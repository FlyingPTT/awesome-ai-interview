### Coding Agents

- SWE-bench: benchmark for software engineering tasks.

- AI teammate studies in software engineering.

### Production Agent Topics

- Tool reliability.
- Long-running task state.
- Permission management.
- Agent evaluation.
- Human-in-the-loop review.

## Interview Takeaways

- Agent performance should be measured by task completion, not only response quality.
- Tool design is part of model quality.
- Evaluation must include failure recovery and safety.
- Enterprise agents need auditability and permission boundaries.
