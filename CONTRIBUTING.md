# Contributing

Thanks for helping improve this handbook.

## Good Contributions

- High-frequency interview questions with strong answers.
- Real-world LLM, RAG, and agent system design cases.
- Debugging notes from production AI systems.
- Paper summaries with engineering takeaways.
- Reusable skills, prompts, and workflows.
- Resume-ready project ideas.

## Content Style

Please keep content:

- Practical.
- Concise.
- Interview-friendly.
- Structured.
- Easy to verify.
