# GitHub 设置指南

用这份指南把仓库设置得更容易被搜索、star 和分享。

## 仓库 Description

推荐 description：

```text
Bilingual handbook for LLM, RAG, AI Agent interviews, skills, papers, and resume-ready projects.
```

更短版本：

```text
Awesome AI interview handbook for LLM, RAG, agents, skills, papers, and projects.
```

## Website

先使用仓库 URL：

```text
https://github.com/FlyingPTT/awesome-ai-interview
```

如果后续开启 GitHub Pages，再改成 Pages URL。

## Social Preview

推荐 social preview 文案：

```text
Awesome AI Interview
LLM • RAG • AI Agents • Skills • Papers • Projects
```

建议视觉风格：

- 简洁深色背景。
- 大标题：`Awesome AI Interview`。
- 副标题：`LLM | RAG | Agents | Skills | Projects`。
- 小标识：`English + 中文`。

## GitHub 主页置顶文案

如果把仓库 pin 到 GitHub profile，可以用：

```text
A bilingual AI interview handbook covering LLM fundamentals, RAG, AI agents, evaluation, skills, paper maps, and runnable resume projects.
```

## Release 策略

建议 tags：

- `v0.1`：initial handbook。
- `v0.2`：bilingual docs and 100 LLM questions。
- `v0.3`：RAG and Agent interview expansions。
- `v0.4`：runnable examples and research maps。
