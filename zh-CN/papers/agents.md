# AI Agent 论文与资源

## 阅读地图

### 基础

- ReAct：语言模型的 reasoning and acting。
- Toolformer：让语言模型使用工具。
- Reflexion：带 verbal reinforcement 的语言 Agent。

### Coding Agents

- SWE-bench：软件工程任务 benchmark。
- Coding agent 采用情况研究。
- 软件工程中的 AI teammate 研究。

### 生产 Agent 主题

- 工具可靠性。
- 长任务状态。
- 权限管理。
- Agent evaluation。
- Human-in-the-loop review。

## 面试要点

- Agent performance 应按 task completion 衡量，而不是只看 response quality。
- Tool design 是模型质量的一部分。
- Evaluation 必须包含失败恢复和安全。
- 企业 Agent 需要 auditability 和 permission boundaries。
