# AI Skills 合集

这个目录收集可复用的 AI skills 和 workflows。Skill 是一种紧凑的指令包，用来帮助 AI Agent 稳定完成可重复任务。

## Skill 分类

- [Coding Agent Skills](coding-agent-skills.md)
- [Agent Skills Catalog](agent-skills-catalog.md)

- [RAG 诊断 Skills](rag-diagnosis-skills.md)
- [热点方向 Skills](hot-domain-skills.md)

## 好 Skill 的要素

一个好的 skill 应定义：

- 目的。
- 输入。
- 工具。
- 工作流。
- 质量标准。
- 失败处理。
- 输出格式。

## Skill 模板

```md
# Skill 名称

## 目的

描述这个可重复任务。

## 输入

- 输入 1。
- 输入 2。

## 工作流

1. 检查上下文。
2. 识别约束。
3. 执行任务。
4. 验证输出。
5. 汇报结果。

## 质量标准

- 要求 1。
- 要求 2。

## 输出格式

定义最终交付物。
```
