# 热点方向 Skills

这些 skills 用于跟踪快速变化的 AI 热点，并把它们转化成面试笔记和项目机会。

## 每周 AI 趋势 Digest Skill

目的：把一周内的 AI 进展总结成可执行的面试和项目笔记。

工作流：

1. 收集模型发布、论文、框架和产品变化。
2. 按主题分组。
3. 识别每个变化为什么重要。
4. 提取面试问题。
5. 提取项目 idea。

输出：

- Top trends。
- Key papers。
- 值得尝试的工具。
- 面试 talking points。
- 项目机会。

## 热点转项目 Skill

目的：把一个热门 AI 方向转换成作品集项目。

工作流：

1. 识别核心技术想法。
2. 选择一个小而真实的 use case。
3. 设计最小架构。
4. 定义指标。
5. 写 resume bullets。
6. 列出面试问题。

示例主题：

- Agentic RAG。
- AI Coding Agents。
- 多模态文档 Agent。
- LLM Evaluation 平台。
- Synthetic Data Pipeline。
