# 研究 Skills

## 论文阅读 Skill

目的：阅读研究论文并提取面试中有价值的洞察。

输入：

- 论文标题或 PDF。
- 目标岗位或主题。

工作流：

1. 识别研究问题。
2. 总结核心方法。
3. 提取关键公式或算法。
4. 与 prior work 比较。
5. 识别局限。
6. 转换成面试 talking points。

输出：

- 一句话总结。
- 方法拆解。
- 优点。
- 缺点。
- 实现注意事项。
- 面试问题。

## 文献综述 Skill

目的：从多篇论文构建主题地图。

工作流：

1. 按问题对论文分组。
2. 按架构对方法分组。
3. 追踪时间线发展。
4. 识别开放问题。
5. 给出阅读顺序。

输出：

- 主题 taxonomy。
- 论文表格。
- 趋势总结。
- 推荐阅读路径。
