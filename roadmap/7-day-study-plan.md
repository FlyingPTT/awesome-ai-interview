# 7-Day AI Interview Study Plan

This plan is designed for candidates preparing for LLM, RAG, and AI agent interviews in one focused week.

## Day 1: Transformer And LLM Basics

Read:

- [LLM Engineer Roadmap](llm-engineer.md)
- [High-Frequency LLM Interview Questions](../interviews/llm-basic.md)
- Questions 1-20 in [100 LLM Interview Questions](../interviews/100-llm-interview-questions.md)

Deliverable:

- Explain attention, positional encoding, decoder-only architecture, and next-token prediction without notes.

## Day 2: Fine-Tuning, Alignment, And Inference

Read:

- Questions 21-35 in [100 LLM Interview Questions](../interviews/100-llm-interview-questions.md)

Practice:

- Compare SFT, RLHF, DPO, LoRA, QLoRA, RAG, and distillation.
- Explain KV cache, prefill/decode, batching, quantization, and speculative decoding.

Deliverable:

- Write a two-minute answer to: "How would you reduce LLM serving cost and latency?"

## Day 3: RAG Engineering

Read:

- [RAG Engineer Roadmap](rag-engineer.md)
- [RAG Interview Notes](../interviews/rag.md)
- [50 RAG Interview Questions](../interviews/50-rag-interview-questions.md)

Practice:

- Design an enterprise knowledge assistant.
- Debug a hallucinated RAG answer.

Deliverable:

- Draw a RAG pipeline and list evaluation metrics for retrieval and generation separately.

## Day 4: AI Agents

Read:

- [AI Agent Engineer Roadmap](ai-agent-engineer.md)
- [AI Agent Interview Notes](../interviews/agent.md)
- [50 AI Agent Interview Questions](../interviews/50-agent-interview-questions.md)

Practice:

- Design a coding agent.
- Explain tool calling, memory, planning, verifier, and guardrails.

Deliverable:

- Write a safe execution policy for an agent that can run shell commands.

## Day 5: Evaluation And Safety

Read:

- [LLM Evaluation Notes](../interviews/eval.md)
- Questions 66-75 in [100 LLM Interview Questions](../interviews/100-llm-interview-questions.md)

Practice:

- Build a regression test plan for a prompt.
- Design LLM-as-a-judge rubrics.
- Explain prompt injection and refusal quality.

Deliverable:

- Create an evaluation table with metrics, datasets, and thresholds.

## Day 6: System Design

Read:

- [LLM System Design Notes](../interviews/system-design.md)
- Questions 76-85 in [100 LLM Interview Questions](../interviews/100-llm-interview-questions.md)

Practice:

- Design an AI customer support assistant.
- Design an LLM evaluation platform.
- Discuss cost, latency, monitoring, and human review.

Deliverable:

- Prepare one full system design answer with architecture, tradeoffs, evaluation, and failure modes.

## Day 7: Resume Project And Mock Interview

Read:

- [Resume-Ready AI Projects](../projects/README.md)
- [Agent Skills Catalog](../skills/agent-skills-catalog.md)
- [Resume Project Template](../templates/resume-project.md)

Practice:

- Pick one project.
- Write resume bullets.
- Run a 45-minute mock interview using the question bank.

Deliverable:

- One polished resume project pitch and one follow-up improvement plan.
