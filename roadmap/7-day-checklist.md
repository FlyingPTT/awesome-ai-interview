# 7-Day AI Interview Checklist

Use this checklist as a one-week sprint for LLM, RAG, and AI agent interviews.

## Day 1: LLM Foundations

- [ ] Explain self-attention from query, key, and value.
- [ ] Explain why Transformers need positional information.
- [ ] Compare encoder-only, decoder-only, and encoder-decoder models.
- [ ] Explain causal masking.
- [ ] Answer questions 1-20 in [100 LLM Interview Questions](../interviews/100-llm-interview-questions.md).

Deliverable:

- [ ] Record or write a two-minute explanation of Transformer attention.

## Day 2: Training, Alignment, And Inference

- [ ] Compare pretraining, SFT, instruction tuning, RLHF, and DPO.
- [ ] Explain LoRA and QLoRA.
- [ ] Explain KV cache, prefill, and decode.
- [ ] Explain batching, quantization, and speculative decoding.
- [ ] Answer questions 21-35 in [100 LLM Interview Questions](../interviews/100-llm-interview-questions.md).

Deliverable:

- [ ] Write a short answer to: "How would you reduce LLM serving latency and cost?"

## Day 3: RAG

- [ ] Draw a RAG pipeline.
- [ ] Explain chunking tradeoffs.
- [ ] Compare dense, sparse, and hybrid retrieval.
- [ ] Explain reranking and context compression.
- [ ] Answer [50 RAG Interview Questions](../interviews/50-rag-interview-questions.md).

Deliverable:

- [ ] Write a debug plan for a hallucinated RAG answer.

## Day 4: AI Agents

- [ ] Explain the model-tool-observation loop.
- [ ] Explain tool schema validation.
- [ ] Compare ReAct and plan-and-execute.
- [ ] Explain memory, state, verifier, and guardrails.
- [ ] Answer [50 AI Agent Interview Questions](../interviews/50-agent-interview-questions.md).

Deliverable:

- [ ] Run [Minimal Agent Framework](../examples/minimal-agent-framework/) and explain each step in its trajectory.

## Day 5: Evaluation And Safety

- [ ] Explain exact, semantic, human, and system evaluation.
- [ ] Explain LLM-as-a-judge and its biases.
- [ ] Explain faithfulness, citation correctness, and refusal quality.
- [ ] Explain prompt injection and tool safety.
- [ ] Run [LLM Eval Harness](../examples/llm-eval-harness/).

Deliverable:

- [ ] Design a release gate for an LLM app.

## Day 6: System Design

- [ ] Design an enterprise knowledge assistant.
- [ ] Design an AI customer support assistant.
- [ ] Design a coding agent.
- [ ] Discuss latency, cost, monitoring, and human review.
- [ ] Review [LLM System Design Notes](../interviews/system-design.md).

Deliverable:

- [ ] Prepare one full system design answer with architecture, tradeoffs, metrics, and failure modes.

## Day 7: Resume Project And Mock Interview

- [ ] Choose one project from [Resume-Ready AI Projects](../projects/README.md).
- [ ] Write two resume bullets.
- [ ] Prepare a three-minute project pitch.
- [ ] Do a 45-minute mock interview using the question banks.
- [ ] List three weaknesses to improve next week.

Deliverable:

- [ ] Finalize one resume-ready AI project story.
