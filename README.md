A practical, interview-oriented handbook for LLM, RAG, AI Agent, and AI research engineer roles.

This repository curates high-frequency interview questions, system design notes, research roadmaps, reusable AI skills, and resume-ready project templates. The goal is to help candidates build a coherent knowledge system instead of memorizing scattered notes.

All interview notes, roadmaps, skills, projects, and templates are maintained in both English and Chinese. Chinese versions are available under [`zh-CN/`](zh-CN/).

- 100 LLM interview questions for fast review and mock interviews.

- LLM fundamentals: Transformer, attention, pretraining, alignment, inference.
- RAG engineering: chunking, embedding, retrieval, reranking, evaluation, observability.
- AI agents: tool use, planning, memory, workflow orchestration, guardrails, evaluation.
- System design: production LLM applications, cost control, latency, reliability.

## How To Use This Repository

1. Pick a target role from `roadmap/`.
2. Review the matching interview notes in `interviews/`.
3. Build one project from `projects/`.
4. Reuse skills from `skills/` to accelerate study and implementation.
5. Turn the project into a resume bullet using `templates/resume-project.md`.
