## 你能在这里找到什么

所有面试笔记、路线图、skills、项目和模板都会维护中英文两个版本。英文版本保留在仓库原始路径，中文版本统一放在 [`zh-CN/`](zh-CN/) 下。

### 面试准备

- 100 道 LLM 面试题，适合快速复习和模拟面试。
- 50 道 RAG 面试题和 50 道 AI Agent 面试题。
- LLM 基础：Transformer、Attention、预训练、对齐、推理。
- RAG 工程：切分、Embedding、检索、重排、评估、可观测性。
- AI Agent：工具调用、规划、记忆、工作流编排、权限和评估。
- 系统设计：生产级 LLM 应用、成本、延迟、可靠性和安全。

### AI Skills 与工作流

- 论文阅读 skill。
- Coding agent skill。
- Agent skills catalog。
- RAG 诊断 skill。
- 面试回答 skill。
- 文献综述 skill。
- 热点方向转项目 skill。

### 研究与热点方向

- Agentic RAG。
- 长上下文记忆。
- 多模态 Agent。
- AI Coding Agent。
- LLM Evaluation。
- 高效推理。
- Synthetic Data。
- Test-time Compute 与推理增强。

### 简历项目

- 生产级 RAG 系统。
- 带 review 和 repair loop 的 Coding Agent。
- LLM Evaluation Harness。
- Minimal Agent Framework。
- 可运行的 minimal agent framework 和 evaluation harness examples。
- RAG mini system、model router 和 coding-agent mini examples。
- RAG eval set 和工程视角论文摘要。
- 论文阅读助手。
- AI 面试模拟器。
- Agent Skill Marketplace 原型。

## 使用方式

1. 从 `zh-CN/roadmap/` 选择目标岗位路线。
2. 阅读 `zh-CN/interviews/` 中对应面试笔记。
3. 从 `zh-CN/projects/` 选择一个项目做作品集。
4. 复用 `zh-CN/skills/` 中的工作流提高学习和实现效率。
5. 用 `zh-CN/templates/resume-project.md` 把项目改写成简历 bullet。
